# 📊 Customer Trends Data Analysis

## 📌 About the Project
This project analyzes customer trends using data analytics techniques.  
It helps in understanding user behavior, campaign performance, and generating insights.

## 🚀 Features
- Data collection and preprocessing
- Campaign performance analysis
- Dashboard creation using Excel
- Insight generation for decision-making

## 🛠️ Tech Stack
- Python
- Pandas
- Excel
- Data Visualization

## 📂 Project Structure
- data/ → dataset files
- notebooks/ → analysis notebooks
- src/ → scripts
- output/ → reports & results

## 📈 What I Did
- Cleaned and prepared data from multiple sources  
- Built reports and dashboards  
- Analyzed user engagement metrics  
- Generated insights to improve campaigns
